# Testify Playground

A collection of sample projects demonstrating how to use the [Testify VSCode extension](https://github.com/felixjb/testify) with various JavaScript/TypeScript test runners.

## About

This repository provides example projects for exploring Testify features, testing edge cases, and learning how to use the extension with different test frameworks including:

- AVA
- Jest
- Mocha
- Node
- Playwright
- Vitest

## Prerequisites

You need to have [Node.js](https://nodejs.org/en/) and [npm](https://www.npmjs.com/) installed on your system to run the examples.

## Getting Started

1. Clone this repository
2. Open a sample subdirectory in VSCode (e.g., `code samples/jest`)
3. Install dependencies: `npm install`
4. Explore the Testify extension features with the sample tests
